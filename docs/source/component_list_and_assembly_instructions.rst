元件清单和组装说明
==========================

你需要先根据清单检查是否有缺少或损坏的组件。如果有任何问题，请联系我们，我们会尽快解决。

请按照PDF上的步骤进行组装。

.. note::

    * 组装前需要购买2节18650电池并充满电，参见 :ref:`关于电池`。
    * Robot HAT不能给电池充电，需要额外购买充电器。
